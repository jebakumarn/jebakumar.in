Project Name : Intel IPU - LAN Performance
Project Duration : APR 2023 - Till Now
Job Designation : Driver Validation Engineer
Organization : Intel - India
Project Description :
  • This project aimed to collect the Intel IPU Host and Accelerater Throughput performance.
Roles and Responsbilites :
  • Reseting BIOS configuration to default values and applying tunned BIOS configuration manually
  • Bringing up Test system: By installing OS (Linux RHEL or Fedora or Rocky linux)
  • Applying performance tunned configuration to Kernel, OS, network adapter and Traffic generator applications based on test requirement.
  • Configure Default node policies / FXP rules between SUT IPU and Client IPU
  • Executing Intel IPU Host and Accelerater Throughput performance test by manullay or using automated scripts using iperf or Netperf or neper traffic generator to collect following performance data sets with CPU utilization of the SUT test system:
    • TCP Traffic Buffer Size = 32 to 65535
    • UDP Traffic Buffer Size = 32 to MAX MTU Size
    • Number of Traffic Threads/flows = 1 to MAX
    • Number of Test ports in Ethernet Adapter = 1 to MAX number of PF ports supported in Network Adapter
    • Traffic Directions = UX (TX , RX) and BX
  • Executing Latency Test by manuall or using automated scripts using netperf tool
  • Analyse collected results, prepare a report for it and share it to team and management
Technologies and Tools Used :
iPerf2, neper and Netperf : used for collecting TCP and UDP Traffic Throughput Performance Data
Bash Shell script :

used for Configuring and bringing up Test system and executed iperf netperf traffic tests

Products Tested :
  • Intel® Infrastructure Processing Unit (Intel® IPU)
