Project Name : Intel IPU - IPDK
Project Duration : Oct 2022 - MAR 2023
Job Designation : Driver Validation Engineer
Organization : Intel - India
Project Description :
  • This project aimed to collect the Throughput performance of Linux Namespace with p4 configuration on Intel IPU Ethernet Network adapters.
Roles and Responsbilites :
  • Reseting BIOS configuration to default values and applying tunned BIOS configuration manually
  • Bringing up Test system: By installing OS (Linux RHEL)
  • Configuring and installing required depedenices, package libs, P4-DPDK rules, microVMs, Namespace VMs to bring up the IPDK test setup
  • Executing Throughput performance test by manullay or using automated scripts using iperf or Netperf traffic generator to collect following performance data sets with CPU utilization of the SUT test system:
    • TCP Traffic Buffer Size = 32 to 65535
    • UDP Traffic Buffer Size = 32 to MAX MTU Size
    • Number of Traffic Threads = 1 to MAX Number of Test system CPU Cores
    • Number of Test ports in Ethernet Adapter = 1 to MAX number of PF ports supported in Network Adapter
    • Traffic Directions = UX (TX , RX) and BX
  • Executing Latency Test by manuall or using automated scripts using netperf tool
  • Analyse collected results, prepare a report for it and share it to team and management
Technologies and Tools Used :
iPerf2 and Netperf : used for collecting TCP and UDP Traffic Throughput Performance Data
Bash Shell script :

used for Configuring and bringing up Test system and executed iperf netperf traffic tests

Products Tested :
  • Intel® Infrastructure Processing Unit (Intel® IPU)
