Project Name : Intel Ethernet - E800 Series Network Adapter - LAN Performance
Project Duration : OCT 2022 - Till Now
Job Designation : Driver Validation Engineer
Organization : Intel - India
Project Description :
This project aimed to collect the LAN Throughput and Latency performance of Intel E700 Series Ethernet Network adapters in test environment and validating against defined KPI.
Roles and Responsbilites :
  • Reseting BIOS configuration to default values and applying tunned BIOS configuration manually
  • Bringing up Test system: By installing OS (Linux RHEL or Windows Server 2019/2022 Data center)
  • Applying performance tunned configuration to Kernel, OS, network adapter and Traffic generator applications based on test requirement.
  • Executing LAN Throughput performance test by manullay or using automated scripts using iperf or Netperf traffic generator to collect following performance data sets with CPU utilization of the SUT test system:
    • TCP Traffic Buffer Size = 32 to 65535
    • UDP Traffic Buffer Size = 32 to MAX MTU Size
    • Number of Traffic Threads = 1 to MAX Number of Test system CPU Cores
    • Number of Test ports in Ethernet Adapter = 1 to MAX number of PF ports supported in Network Adapter
    • Traffic Directions = UX (TX , RX) and BX
  • Executing Latency Test by manuall or using automated scripts using netperf tool
  • Analyse collected results, prepare a report for it and share it to team and management
Technologies and Tools Used :
iPerf2 : used for collecting TCP and UDP LAN Throughput Performance Data
netperf : used for collecting TCP / UDP LAN Throughput and Latency Performance Data
Bash Shell and python script :

used for collecting LAN Throughput Performance data on Linux (RHEL) OS

used parmiko, subprocess, os, openpyxl python modules to automate End to End Test
BAT and Powershell script : used for collecting LAN Throughput Performance data on Windows Server OS
Challenges :
  • Collecting One set of LAN performance test result by manually takes ~ 12 to 14 hours. If each traffic test executes for 30 seconds
Achievements :
  • Using Shell, BAT, powershell and Python Automated LAN performance test.
  • Reduced execution time to Half without manuall intervention using standalone automation script
  • Automated script will take care below things
    • Appplying Performance Tunning configurations to Test systems and Test Network Adapters
    • Installing Traffic Generator Tools and other required depedencies packages, apps, libs
    • Collecting Test System information - OS details, Network Adapter details, Kernel details, hardware information and other basic informations
    • Testing connectivity between SUT to each client test system
    • If connectivity check passed, start execuing LAN performance test with multiple scalable parameters
    • Each test will collect Throughput data and CPU utilization data.
    • Collected data will redirect to Excel CSV file
Products Tested :
  • Intel® Ethernet Network Adapter E810-2CQDA1 / CQDA2
  • Intel® Ethernet Network Adapter E810-XXVDA2
  • Intel® Ethernet Network Adapter E810-XXVDA4
